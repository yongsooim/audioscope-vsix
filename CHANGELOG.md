# Changelog

All notable changes to this project will be documented in this file.

## 1.0.0 - 2026-04-01

- Initial public release of `audioscope`
- Added a read-only custom audio viewer for `.wav`, `.wave`, `.mp3`, `.ogg`, `.oga`, `.flac`, `.m4a`, `.aac`, `.opus`, `.aif`, and `.aiff`
- Added synchronized waveform, spectrogram, looping, follow playback, timeline overview, playback speed, metadata, and loudness inspection
- Bundled embedded FFmpeg WebAssembly tools for metadata probing and decode fallback
- Added release verification docs and packaging checks for VSIX publishing
